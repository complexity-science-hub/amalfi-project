# AMALFI Project Website

This is the official website for the AMALFI project, built with Jekyll using the Minimal Mistakes theme.

## Local Development

To run the site locally:

```bash
bundle install
bundle exec jekyll serve
```

The site will be available at `http://localhost:4000/amalfi-project/`

## Deployment

This site is published using GitHub Pages at: https://complexity-science-hub.github.io/amalfi-project/

## Theme

This site uses the [Minimal Mistakes Jekyll theme](https://mmistakes.github.io/minimal-mistakes/).
